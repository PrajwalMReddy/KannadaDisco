# Question Words

### Kannada also has a set of question words. These are given below.

|  English  | Kannada | Transcription |
|:---------:|:-------:|:-------------:|
|    Who    |  ಯಾರು   |     yāru      |
|   What    |   ಏನು   |      ēnu      |
|   When    |  ಯಾವಾಗ  |    yāvāga     |
|   Where   |  ಎಲ್ಲಿ  |     elli      |
|    Why    |  ಯಾಕೆ   |     yāke      |
|    How    |  ಹೇಗೆ   |     hēge      |
| How Much  |  ಎಷ್ಟು  |     eṣṭu      |
|   Which   |   ಯಾವ   |     yāva      |
| Which One | ಯಾವಾದು  |    yāvādu     |
